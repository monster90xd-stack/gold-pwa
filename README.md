# GCC Gold Calculator / حاسبة الذهب للخليج

This repository contains a small PWA (HTML/CSS/JS) for gold prices + karat (kirat) calculator.

## Deploy (GitHub Pages)
1. Repo Settings → Pages
2. Source: Deploy from a branch
3. Branch: main, folder: / (root)
4. Visit the Pages URL once it is built.

## Configure
Edit `app.js` and set `GOLDAPI_KEY`.
